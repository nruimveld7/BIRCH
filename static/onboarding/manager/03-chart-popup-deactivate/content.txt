Deactivate Chart
When a chart is no longer needed, it should be deactivated.

When deactivating a chart, the Manager that clicked "Deactivate" will be removed from the chart.
If a Manager has deactivated the chart accidentally, and was not the last Manager, another Manager will have to grant Manager level access to that user again.

A chart is only fully deactivated when every Manager has clicked "Deactivate"

Only deactivate when you are confident the chart should no longer be available.

When the last Manager attempts to deactivate the chart, they will receive this confirmation popup.
WARNING: Full chart deactivation is irreversible.
