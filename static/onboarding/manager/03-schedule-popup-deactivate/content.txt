Deactivate Schedule
When a schedule is no longer needed, it should be deactivated.

When deactivating a schedule, the Manager that clicked "Deactivate" will be removed from the schedule.
If a Manager has deactivated the schedule accidentally, and was not the last Manager, another Manager will have to grant Manager level access to that user again.

A schedule is only fully deactivated when every Manager has clicked "Deactivate"

Only deactivate when you are confident the schedule should no longer be available.

When the last Manager attempts to deactivate the schedule, they will receive this confirmation popup.
WARNING: Full schedule deactivation is irreversible.
